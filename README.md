# SiteGrauna

Vitrine digital da **Graúna Moda** — moda feminina em Santa Maria/RS.

## Stack

- React + TypeScript + Vite
- Tailwind CSS
- Framer Motion + Lenis
- Three.js / React Three Fiber

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
